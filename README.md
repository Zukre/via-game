# VIA — денежная настольная игра (онлайн-сервер)

Общий сервер синхронизации для игры Via: все игроки видят одни бланки в реальном времени.

## Локально
```
python via_server.py
```
Откроется `http://localhost:8080/via.html`.

## Облако (Render.com — бесплатно, постоянная ссылка)
Сервер уже готов к облаку: порт берётся из `$PORT`, слушает `0.0.0.0`.
Деплой описан в `render.yaml`. Старт: `python via_server.py`.
Зависимостей нет — только стандартная библиотека Python.
